# Project-Group-3

# Requirements
Backend:
- go to terminal
- cd backend
- pip install -r requirements.txt

Frontend:
- go to terminal
- cd frontend
- npm install


# HOW TO RUN
Run the backend:
- go to terminal
- cd backend
- python manage.py runserver or python3 manage.py runserver

Run the frontend:
- go to terminal
- cd frontend
- npm start
